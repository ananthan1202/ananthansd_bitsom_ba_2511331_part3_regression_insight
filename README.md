# ananthansd_bitsom_ba_2511331_part3_regression_insight
Regression Insights Project

Project Overview

This project analyzes the factors affecting monthly sales using simple and multiple linear regression models in Microsoft Excel. The objective is to identify the variables that have the strongest association with sales and provide business recommendations based on statistical evidence.

---

Objectives

- Perform simple linear regression analysis.
- Perform multiple linear regression analysis.
- Compare regression models.
- Analyze residuals.
- Develop regression equations.
- Provide business recommendations supported by statistical analysis.

---

Dataset

The dataset contains 320 observations and includes the following variables:

- Monthly Sales (Dependent Variable)
- Marketing Spend
- Footfall
- Inventory Availability (%)
- Region (East, North, South, West)

---

Tools Used

- Microsoft Excel
- Data Analysis ToolPak
- Linear Regression

Key Findings

- Marketing Spend has a strong positive relationship with monthly sales.
- Footfall also contributes positively to sales.
- Inventory Availability significantly improves sales performance.
- South and West regions perform better than the reference region.
- North region was not statistically significant.
- The Multiple Regression model provided the best performance with an R² of approximately 0.8114, explaining over 81% of the variation in monthly sales.

---

Conclusion

The Multiple Regression model was selected as the final model because it provides the highest explanatory power and incorporates multiple business factors simultaneously. The results suggest that improving marketing investment, increasing customer footfall, and maintaining high inventory availability are the most effective strategies for increasing monthly sales.
